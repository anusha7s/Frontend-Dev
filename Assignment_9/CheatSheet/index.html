Responsive CSS Cheat Sheet

A quick summary of key responsive CSS concepts and examples

CSS Filters

Example: filter: grayscale(80%) brightness(1.2);
Filters adjust visual effects such as blur, brightness, and color tone.

Media Queries

Example:
@media (max-width: 700px) { body { font-size: 14px; } }
Used to apply styles based on device width or orientation.

Responsive Design

Technique: Use relative units like %, em, or vw for widths and spacing.
Ensures your layout adapts across screens and devices.

Responsive Typography

Example: font-size: clamp(1rem, 2vw, 2rem);
Text scales smoothly based on viewport width for readability.
