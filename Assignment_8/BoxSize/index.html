Box-Sizing Comparison Page

Default: content-box

In content-box (default), the width property applies only to the content. Padding and border are added outside, increasing the total box size.

box-sizing: border-box;

In border-box, the width includes content, padding, and border. The total size stays fixed, making layouts easier to control.
